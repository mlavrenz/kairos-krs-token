# Kairos Token (KRS)

> Every Transaction Is a Mission.

## Overview
Kairos (KRS) is a fixed-supply ERC-20 utility token deployed on Polygon Mainnet. 
Every transfer automatically distributes 4% to mission-aligned wallets.

## Token Details
| Field | Value |
|-------|-------|
| Name | Kairos |
| Symbol | KRS |
| Network | Polygon Mainnet |
| Total Supply | 100,000,000 KRS (Fixed) |
| Decimals | 18 |
| Contract | 0x9d6dffc40c5a5faa7fa581269cb9f3055bcbf2ce |

## Fee Structure
| Wallet | Fee | Purpose |
|--------|-----|---------|
| Mission | 2% | Humanitarian funding |
| Staking | 1% | Staking rewards |
| Treasury | 1% | Operations |
